 NORMAL MAP and TEXTURES

 Normal Map -- >
 When the model is exported it cn have all the details and surface ips and downs in model itslef but this will make model file bigger and make it take longer time to load , which we don't want

So we just add the essential details in the model  and other details are added through "Normal maps" which are images ( mostly in pn format )

We load the model and after loading we get the details from normal map and apply it on the model




Textures - 
All the textures ( appearanch and feel of model surface)
are not added in the model itself , but we create image of it too , just like normal maps and add the details from that image to model, after loading the model 
( colour texture and matcap texture)

Both normal map and texture are created by the model artist
